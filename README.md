# Evolutionary Algorithms Notebooks

This repository contains two compact Jupyter notebooks for demonstrating core evolutionary computation workflows.

## Included Notebooks

- `genetic-algorithm-notebook.ipynb`
- `genetic-programming-notebook.ipynb`

## What They Cover

- binary genetic algorithms
- genetic programming with compact program representations
- experiment-oriented notebook workflows
- visual and tabular result inspection inside notebooks

## Repository Scope

The public repository is intentionally minimal. It keeps the notebooks themselves and removes unrelated local files and generated experiment outputs.
